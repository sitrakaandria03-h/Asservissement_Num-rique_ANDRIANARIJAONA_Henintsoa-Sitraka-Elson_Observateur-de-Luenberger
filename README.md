#Description du projet
Ce projet porte sur la modélisation d'un observateur de Luenberger sur un système masse-ressort. L'objetcif est de voir l'efficacité de l'observateur à suivre correctement le comportement interne du système masse-ressort sans avoir recours à des capteurs.

#Instruction d'excécution
- Lancez le script "parametre.m" dans MATLAB pour l'initialisation des paramètres.
-Ouvrez le fichier simulink "simulink_observateur_luenberger_masse_ressort.slx".
-Lancez la simulation et ouvrez le bloc scope pour visualiser (il y a deux blocs scope, l'un c'est la réponse de l'observateur qui suit le comportement du système masse-ressort et l'autre c'est seulement le comportement du système masse-ressort)
